---------------------2. JavaBean class-----------------------------------

A JavaBean is a special type of POJO that follows some rules.

-class must be public
-Must have a public no-argument constructor
-Fields should be private
-Provide public getter and setter methods
-Usually implements Serializable

class Student{

------------pojo vs bean class---------------

| POJO                         | JavaBean                                         |
| ---------------------------- | ------------------------------------------------ |
| Simple Java object           | Special type of POJO                             |
| No strict rules              | Must follow rules                                |
| Can have any constructor     | Requires no-arg constructor                      |
| Fields can be public/private | Fields should be private                         |
| Getter/setter optional       | Getter/setter required                           |
| Serializable optional        | Usually Serializable                             |
| Used everywhere              | Used in frameworks (Spring, Hibernate, JSP etc.) |
