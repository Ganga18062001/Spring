
=> A JavaBean is a java class that follows the following rules : -

1)It should have private properties
2)It should provide public getter and setter methods to get and set the values in properties
3)It should have public no-arg constructor or default constructor

------------java bean object -------------------------------------
-java bean object ia an instance of java bean class.
-It is reusable ,encapsulated java componenet with properties , getter and setters,designed for
easy integration and manipulation
