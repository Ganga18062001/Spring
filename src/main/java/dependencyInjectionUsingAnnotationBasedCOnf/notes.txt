---------dependency injection-------------

-It is a design pattern.

-Its main task is to "inject the dependency" means
inject one object into another object.

-It is used to achieve "loose coupling" in java.

-we can achieve dependency injection by 2 ways :-
1)setter method DI
2)constructor DI

2)Tightly couple

-Tightly coupled classes have a strong dependency.

-One class directly depends on another.

syntax :-

//start the engine
}
}

public class Car{
